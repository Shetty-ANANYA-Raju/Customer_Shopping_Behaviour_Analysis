# 🛍️ Customer Behavior Analytics Dashboard

### End-to-End Data Analytics Project using Python, MySQL, SQL & Power BI

This project focuses on analyzing customer purchasing behavior to uncover patterns, preferences, and business opportunities. Using Python and Pandas for data cleaning, MySQL for data storage, SQL for analysis, and Power BI for visualization, the project transforms raw customer transaction data into actionable business insights.

The dashboard helps businesses understand:

* Customer purchasing patterns
* Revenue contribution by customer segments
* Product category preferences
* Customer loyalty and repeat purchase behavior
* Subscription and payment method trends

By combining data engineering, SQL analysis, and business intelligence techniques, this project demonstrates the complete analytics workflow from raw data to executive-level insights.
